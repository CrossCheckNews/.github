# CrossCheckNews

CrossCheckNews는  
같은 이슈를 서로 다른 국가와 성향의 언론이  
어떻게 다르게 보도하는지를 비교할 수 있는 서비스입니다.

단순 뉴스 요약 서비스가 아니라  
**"뉴스 관점 비교 서비스"**를 목표로 합니다.

사용자는 다양한 시각을 비교하며  
정보를 더 입체적으로 이해할 수 있습니다.

---

## 🎯 프로젝트 목표

AI Agent를 활용하여 단 기간 내에 설계부터 구축까지 1인으로 개발했습니다.
순수 소요시간 48시간을 목표로 진행하고 있습니다.

---

## 🧩 핵심 기능

- 언론사별 뉴스 헤드라인 수집 (RSS 기반)
- 동일 주제 기사 클러스터링(방법은 다시 업데이트 예정)
- AI 기반 이슈 요약 (1~2줄)
- 언론사별 헤드라인 비교

---

## 🌍 지원 범위

- 국가: 한국 / 미국
- 성향: 진보 / 보수

---

## 🛠 Tech Stack

- Backend: Java, Spring Boot, JPA
- Frontend: React, Vite
- AI: GPT / Gemini / Claude Code, Cursor
